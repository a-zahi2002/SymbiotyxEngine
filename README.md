# SymbiotixEngine

A multimodal human-computer interaction engine integrating gesture capture, grammar processing, intensity analysis, and Unity-based visualization.

## Project Structure

```
SymbiotixEngine/
├── data/          — Raw, processed, and augmented datasets
├── backend/       — Core processing modules and server
├── unity/         — Unity project files and UI
├── docs/          — Documentation
├── tests/         — Test suites
└── scripts/       — Utility and automation scripts
```
